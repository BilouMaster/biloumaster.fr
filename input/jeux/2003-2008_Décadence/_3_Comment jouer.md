Sur ordinateur, utilisez les touches du clavier :

* Touches directionnelles `←↑↓→` pour **se déplacer**
* Touche `Entrée`, `Espace` ou `Z` pour **valider**
* Touche `Échap`, `X`, `C`, `V`, `B` ou `N` pour **afficher le menu** ou **annuler**
* Touche `shift` pour **courir**
* `F` accélère le jeu x3, `G` x10

**Sur navigateur, en plein écran... n'utilisez pas `Échap` !**

Ça quitterait le plein écran... utilisez les autres touches citées ! **;)**
